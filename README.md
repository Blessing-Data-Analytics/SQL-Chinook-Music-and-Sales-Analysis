# SQL-Chinook-Music-and-Sales-Analysis
This project demonstrates practical SQL skills by analyzing two relational datasets using Microsoft SQL Server Management Studio (SSMS). The objective was to import data, explore database structures, write SQL queries, solve business problems, and apply query optimization techniques.

------

## Overview
This project demonstrates SQL analysis using Microsoft SQL Server Management Studio (SSMS). Two datasets were analyzed to answer business questions and showcase SQL proficiency.

## Datasets
- Chinook Database
- Sample Sales Dataset

## Tools
- SQL Server Management Studio
- GitHub

## SQL Concepts
- SELECT
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- Aggregate Functions
- Subqueries
- Window Functions
- Indexing

## Business Questions
- Who are the highest-spending customers?
- Which products generate the most revenue?
- What are the monthly sales trends?
- Which countries contribute the highest revenue?

## Files
- Chinook_Queries.sql
- Sales_Queries.sql
- SQL_Project_Report.pdf
- Images(screenshots of queries)

## Key Insights
- Revenue is concentrated among a small percentage of customers.
- Product performance varies significantly across categories.
- SQL joins provide comprehensive insights across related tables.
- Window functions simplify ranking and trend analysis.
